#Sales-Data-Analysis-Excel-Dashboard

This project is an end-to-end Excel dashboard designed  to demonstrate key analytical and visualization skills using Excel. It showcases how raw sales data can be transformed into actionable insights using Excel's powerful features like pivot tables, charts, and slicers.
